

## The seven layers of the OSI model are:
 - 7. Application layer
  - PDU: DATA
  - 
 - 6. Presentation layer
  - PDU: DATA
 - 5. Session layer
  - PDU: DATA
 - 4. Transport layer
  - PDU: Segments or Datagrams, depending on protocol used.
 - 3. Network layer
  - PDU: Packets
 - 2. Data link layer
  - PDU: Frames
 - 1. Physical layer
  - PDU:Bits
